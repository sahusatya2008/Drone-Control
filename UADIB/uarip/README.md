# UARIP Mode (Universal Autonomous Robotics Intelligence Platform)

UARIP is an additional mode layered into UADIB without replacing existing drone flows.

## Features in this scaffold

- Protocol reverse-engineering workflow prototype (`uarip/protocol_ai`)
- Capability graph construction (`uarip/robot_graph`)
- Dynamic UI panel generation (`uarip/ui_generator`)
- Swarm coordination step engine (`uarip/swarm_ai`)
- Simulation provider registry (`uarip/simulation`)
- Security anomaly scoring (`uarip/security`)

## API namespace

All UARIP routes are namespaced under `/uarip/*`.
